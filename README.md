## Description📖
#### The exercises cover

 ##### Input

    Leia: prompt, readline

 ##### Output

#### Estruturas de Condição
- SE: `if`
- Senão: `else`
- Encadeado: `if...else if...else`
- Composto: `if...if`
- Escolha: `switch`

#### Estruturas de Repetição
- Enquanto: `while`
- Repita: `do...while`
- Para: `for`
- For...Of: `for (let item of array) {...}`
- Try...Catch: `for (let key in object) {...}`

#### Operadores Lógicos
- E: `&&`
- Ou: `||`
- Não: ``!``
- Diferente: ``!==``
- Igual: ``===``
